Стили

Стили отвечают за внешний вид компонентов страницы.

Стили могут задаваться в:

  1. Внутри тега элемента
  2. На странице в теге <style>
  3. В файле css

Внутри тега элемента

<p style="color: darkorchid; font-size: x-large"> — спортивная командная игра с мячом, в которой мяч забрасывают руками в кольцо соперника.</p>

Стили отделяются друг от друга ;.

На странице в теге <style>

<head>
    ...
    <style>
        .main-definition {
            color: blue;
        }
    </style>
</head>
<body>
<h1>Баскетбол</h1>
<p class="main-definition"> — спортивная командная игра с мячом, в которой мяч забрасывают руками в кольцо соперника.</p>
...

В файле css

index.html

<head>
    ...
    <link rel="stylesheet" href="style.css">
</head>
<body>
<h1>Баскетбол</h1>
<p class="main-definition"> — спортивная командная игра с мячом, в которой мяч забрасывают руками в кольцо соперника.</p>
...

style.css

.main-definition {
    color: red;
}

Файлы в html подключаются при помощи тега <link>, в котором указывается тип связи rel="stylesheet" и путь до файла href="style.css".

Данные способы расположены в порядке уменьшения приоритета.

Для принудительного установления стиля используют ключевое слово !important, которое добавляется после значения свойства.

.main-definition {
    color: red !important;
}

Стили задаются при помощи свойств и их значений, разделённых :.

Элементы, к которым применяются стили, задаются при помощи селекторов.

Теги

  1. <style></style>
    Контейнер для описания стилей страницы.
  2. <link>
    Подключение внешних файлов к странице.

Атрибуты

  1. class
    Задаёт класс элементу.
    У элемента может быть несколько классов. Каждый класс отделяется пробелом.
  2. href
    Задаёт путь до файла.
  3. rel
    Задаёт тип подключаемого файла.

Свойства

  1. color
    Задаёт цвет текста. Цвет можно задавать несколькими способами: ключевыми словами (например, black, red), функцией rgb(), шестнадцатеричным кодом, моделью HSL и HWB.

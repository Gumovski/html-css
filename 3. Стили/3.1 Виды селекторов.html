Виды селекторов

1. Универсальный селектор *

Выбирает все элементы на странице.

2. Селектор элемента тип_элемента { }

Выбирает все элементы какого-либо типа.

div {
    
}

3. Селектор класса .имя_класса { }

Выбирает все элементы какого-либо класса.

4. Селектор индентификтора (id) #id { }

Выбирает элемент с данным id.

5. Селектор атрибута

Выбирает элементы, которые удовлетворяют заданной структуре атрибута.

  • [атрибут] элементы, содержащие данный атрибут.
  • тип_элемента[атрибут] элементы данного типа, содержащие указанный атрибут.
  • [атрибут="значение"] элементы, содержащие указанный атрибут с определённым значением.
  • [атрибут~="значение"] элементы, частично содержащие данное значение, разделённое пробелами.
  • [атрибут|="значение"] элементы, значение атрибута которых начинается с данного значения или с фрагмента значения, после которого идёт дефис (-).
  • [атрибут^="значение"] элементы, значение атрибута которых начинается с данного значения
  • [атрибут$="значение"] элементы, значение атрибута которых заканчивается данным значением.
  • [атрибут*="значение"] элементы, значение атрибута которых содержит данное значение в любом месте.

6. Селектор псевдокласса

Выбирает элемент, находящийся в определённом состоянии.

Селектор Состояние
:linkНе посещённая ссылка
:visitedПосещённая ссылка
:hoverЭлемент, на который наведён курсор мыши
:focusИнтерактивный элемент, к которому перешли при помощи клавиатуры или активировали мышью
:aсtiveАктивный элемент
:validПоле ввода, содержимое которых прошло проверку на соответствие типу
:invalidПоле ввода, содержимое которых не прошло проверку на соответствие типу
:enabledВсе активные поля формы
:disabledВсе элементы форм, находящиеся в заблокированном состоянии
:checkedЭлементы формы, выбранные пользователем
:targetТекущий активный элемент, выделенный с помощью якоря #
:emptyЭлементы, у которых внутри нет других элементов и контента
:lang(язык)Элементы, у которых указан атрибут lang, а значение соответствует выбранному языку
:read-onlyЭлементы формы, к которым применено свойство readonly
:requiredЭлементы формы, к которым применено свойство required
:read-writeЭлементы формы, к которым не применено свойство readonly
:out-of-rangeЭлементы формы, у которых значение выходит за указанный диапазон
:in-rangeЭлементы формы, у которых значение не выходит за указанный диапазон
:not(селектор)Элементы, которые не соответствуют заданному селектору
:nth-child(n)Дочерние элементы, кратные n
:nth-last-child(n)Элемент, являющийся последним кратным n элементом родителя
:nth-last-of-type(n)Элемент, являющийся последним кратным n элементом родителя того же типа
:nth-of-type(n)Элемент, кратные n, того же типа, что и родитель
:only-of-typeЭлементы, которые являются единственными наследниками родителя того же типа
:only-childЭлементы, которые являются единственными наследниками родителя
:optionalЭлементы формы, не содержащие атрибута required
:rootВыбирает корневой элемент страницы

Использование nth селекторов

Параметром таких селекторов является указание номера элементов внутри какого-либо родителя.
n - указывает число
odd, even - чётные или нечётные элементы
an + b - формула, задающая смещение элементов. a и b - числа.

7. Селектор псевдоэлемента

Используются для добавления содержимого к элементам.

Селектор Описание
::afterДобавляет какое-либо содержимое после элемента
::beforeДобавляет какое-либо содержимое перед элементом
::first-letterВыбирает первый символ элемента
::first-lineВыбирает первую строку текста элемента
::markerВыбирает маркеры списка
::selectionВыбирает ту часть элемента, которая выделена пользователем

8. Комбинирование селекторов

8.1 Селектор нескольких элементов (,)

8.2 Селектор потомка (пробел)
Если необходимо указать на потомков (всех) внутри элемента, эти селекторы указывают через пробел.
div p {} - все p, находящие внутри div

8.3 Дочерний селектор (>)
Выбирает элементы, являющиеся прямыми потомками родительского элемента. Т.е. элементы, являющиеся потомками на первом уровне иерархии.

8.4 Селектор соседнего элемента (+)
Выбирает элемент, который идёт непосредственно после другого конкретного элемента.

8.5 Общий родственный элемент (~)
Выбирает все элементы, которые являются потомками на одном уровне иерархии.

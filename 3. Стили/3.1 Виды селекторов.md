# Виды селекторов

1. Универсальный селектор ```*```\
Выбирает все элементы на странице.
2. Селектор элемента ```тип_элемента { }```\
Выбирает все элементы какого-либо типа.
```css
div {
    
}
```
3. Селектор класса ```.имя_класса { }``` \
Выбирает все элементы какого-либо класса.
4. Селектор индентификтора (id) ```#id { }```
Выбирает элемент с данным id.
5. Селектор атрибута\
Выбирает элементы, которые удовлетворяют заданной структуре атрибута.\
    5.1 ```[атрибут]``` элементы, содержащие данный атрибут.\
    5.2 ```тип_элемента[атрибут]``` элементы данного типа, содержащие указанный атрибут.\
    5.3 ```[атрибут="значение"]``` элементы, содержащие указанный атрибут с определённым значением.\
    5.4 ```[атрибут~="значение"]``` элементы, частично содержащие данное значение.\
    5.5 ```[атрибут|="значение"]``` элементы, значение атрибута которых начинается с данного значения.\
    5.6 ```[атрибут^="значение"]``` элементы, значение атрибута которых начинается с данного значения, но не равняется ему.\
    5.7 ```[атрибут$="значение"]``` элементы, значение атрибута которых заканчивается данным значением.\
    5.8 ```[атрибут*="значение"]``` элементы, значение атрибута которых содержит данное значение в любом месте.\

### Использование nth селекторов

Параметром таких селекторов является указание номера элементов внутри какого-либо родителя.\
`n` - указывает число \
`odd`, `even` - чётные или нечётные элементы \
`an + b` - формула, задающая смещение элементов. `a` и `b` - числа.

8. Комбинирование селекторов\
8.1 Селектор нескольких элементов (`,`)\
8.2 Селектор потомка (`пробел`)\
    Если необходимо указать на потомков (всех) внутри элемента, эти селекторы указывают через пробел.\
    `div p {}` - все p, находящие внутри div\
8.3 Дочерний селектор (`>`)\
    Выбирает элементы, являющиеся прямыми потомками родительского элемента. Т.е. элементы, являющиеся потомками на первом уровне иерархии.\
8.4 Селектор соседнего элемента (`+`)\
    Выбирает элемент, который идёт непосредственно после другого конкретного элемента.\
8.5 Общий родственный элемент (`~`)\
    Выбирает все элементы, которые являются потомками на одном уровне иерархии.\

Списки

В HTML существует три типа списков: маркированные, нумерованные и списки определений.

Маркированный список <ul>

Маркированный список создаётся с помощью тега <ul> (unordered list). Каждый элемент списка оборачивается в тег <li> (list item).

<ul>
    <li>Первый элемент</li>
    <li>Второй элемент</li>
    <li>Третий элемент</li>
</ul>

Нумерованный список <ol>

Нумерованный список создаётся с помощью тега <ol> (ordered list). Каждый элемент списка также оборачивается в тег <li>.

Атрибуты <ol>

  1. type
    тип нумерации. Может принимать следующие значения:
    • 1 - арабские цифры (по умолчанию)
    • A - заглавные латинские буквы
    • a - строчные латинские буквы
    • I - заглавные римские цифры
    • i - строчные римские цифры
  2. start
    начальное значение нумерации (число)
  3. reversed
    обратная нумерация (от большего к меньшему)
<ol>
    <li>Первый элемент</li>
    <li>Второй элемент</li>
    <li>Третий элемент</li>
</ol>

<ol type="A">
    <li>Элемент A</li>
    <li>Элемент B</li>
    <li>Элемент C</li>
</ol>

<ol start="5">
    <li>Элемент 5</li>
    <li>Элемент 6</li>
    <li>Элемент 7</li>
</ol>

<ol reversed>
    <li>Элемент 3</li>
    <li>Элемент 2</li>
    <li>Элемент 1</li>
</ol>

Список определений <dl>

Список определений создаётся с помощью тега <dl> (definition list). Используется для представления пар термин-определение.

  • <dt> (definition term) - термин
  • <dd> (definition description) - определение термина
<dl>
    <dt>HTML</dt>
    <dd>Язык разметки гипертекста</dd>
    
    <dt>CSS</dt>
    <dd>Каскадные таблицы стилей</dd>
    
    <dt>JavaScript</dt>
    <dd>Язык программирования для веб-разработки</dd>
</dl>

Вложенные списки

Списки могут быть вложенными друг в друга. Для этого внутри элемента <li> размещается новый список.

<ul>
    <li>Фрукты
        <ul>
            <li>Яблоко</li>
            <li>Банан</li>
            <li>Апельсин</li>
        </ul>
    </li>
    <li>Овощи
        <ul>
            <li>Морковь</li>
            <li>Помидор</li>
            <li>Огурец</li>
        </ul>
    </li>
</ul>

CSS-свойства для списков

Свойство list-style-type

Определяет тип маркера или нумерации для элементов списка.

Возможные значения для маркированных списков:

  • disc - закрашенный круг (по умолчанию)
  • circle - незакрашенный круг
  • square - квадрат
  • none - без маркера

Возможные значения для нумерованных списков:

  • decimal - арабские цифры (по умолчанию)
  • decimal-leading-zero - арабские цифры с нулём впереди (01, 02, 03...)
  • lower-alpha - строчные латинские буквы (a, b, c...)
  • upper-alpha - заглавные латинские буквы (A, B, C...)
  • lower-roman - строчные римские цифры (i, ii, iii...)
  • upper-roman - заглавные римские цифры (I, II, III...)
  • lower-greek - строчные греческие буквы (α, β, γ...)
  • none - без нумерации
/* Маркированный список с квадратными маркерами */
ul {
    list-style-type: square;
}

/* Нумерованный список с римскими цифрами */
ol {
    list-style-type: upper-roman;
}

/* Список без маркеров */
ul {
    list-style-type: none;
}

Свойство list-style-position

Определяет положение маркера относительно элемента списка.

Возможные значения:

  • outside - маркер находится за пределами блока элемента списка (по умолчанию)
  • inside - маркер находится внутри блока элемента списка
/* Маркер внутри блока */
ul {
    list-style-position: inside;
}

/* Маркер снаружи блока */
ul {
    list-style-position: outside;
}

Свойство list-style-image

Позволяет использовать изображение в качестве маркера списка.

Возможные значения:

  • none - без изображения (по умолчанию)
  • url(путь) - путь к изображению
/* Использование изображения в качестве маркера */
ul {
    list-style-image: url('marker.png');
}

/* Отключение изображения */
ul {
    list-style-image: none;
}

Свойство list-style

Короткая запись для одновременной установки list-style-type, list-style-position и list-style-image.

Синтаксис:

list-style: [type] [position] [image]

Порядок значений не имеет значения. Можно указать одно, два или все три значения.

/* Полная запись */
ul {
    list-style: square inside url('marker.png');
}

/* Только тип маркера */
ul {
    list-style: circle;
}

/* Тип и позиция */
ul {
    list-style: decimal inside;
}

/* Удаление всех стилей списка */
ul {
    list-style: none;
}

Дополнительные CSS-свойства

Для более гибкой стилизации списков можно использовать другие CSS-свойства:

/* Отступы и интервалы */
ul {
    padding-left: 40px; /* отступ слева */
    margin: 20px 0; /* внешние отступы */
}

li {
    margin-bottom: 10px; /* отступ между элементами */
    padding: 5px; /* внутренний отступ */
}

/* Стилизация с использованием псевдоэлементов */
ul {
    list-style: none;
    padding-left: 0;
}

li::before {
    content: "→ "; /* пользовательский маркер */
    color: #ff6600;
    font-weight: bold;
}

/* Счётчики для кастомной нумерации */
ol {
    list-style: none;
    counter-reset: custom-counter;
}

ol li {
    counter-increment: custom-counter;
}

ol li::before {
    content: counter(custom-counter) ". ";
    color: #0066cc;
    font-weight: bold;
}

Ссылки

Необходимы для перехода на другую страницу или для перемещения внутри страницы.

Для создания ссылки используется тег <a></a>.

Атрибуты:

href
url внешнего ресурса, путь к документу внутри проекта или якорь

target
Окно открытия ссылки
Значения:
_blank - Открытие ссылки в новом окне
_self - Открытие ссылки а текущем окне

download
Вместо перехода по ссылке, предлагает скачать файл.

title
Добавляет к ссылке всплывающую подсказку.

<p>Баскетбол входит в программу <a href="olympic.html">Олимпийских игр</a> с 1936 года (изобретатель игры
<a title="Страница в википедии" target="_blank" href="https://ru.wikipedia.org/wiki/Нейсмит,_Джеймс">Джеймс Нейсмит</a>
был там в качестве гостя).
Регулярные чемпионаты мира по баскетболу среди мужчин проводятся с 1950 года, среди женщин — с 1953 года,
а чемпионаты Европы — с 1935 года.</p>

Переход внутри документа

Для перехода внутри документа используется якорь - элемент помеченый атрибутом id.
Для перехода по якорям в атрибуте href указывают #id.

<a href="#about"></a>
<h1 id="about"></h1>

Теги

  1. <a></a>
    Гиперссылка

Атрибуты

  1. id
    Уникальный идентификатор элемента. Не может повторяться на странице.
  2. href
    Ссылка на ресурс, файл или якорь.
  3. target
    Метод открытия ссылки.
  4. download
    Указание браузеру на то, что документ нужно не открывать, а скачивать.
  5. title
    Всплывающая подсказка ссылки.

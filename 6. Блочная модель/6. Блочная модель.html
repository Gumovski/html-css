Блочная модель (Box Model)

Каждый элемент на веб-странице представляет собой прямоугольный блок. Блочная модель CSS описывает, как рассчитываются размеры этого блока, включая его содержимое, отступы, рамки и поля.

Блочная модель состоит из четырех частей (от центра к краям):

  1. Content (Содержимое) — текст, изображения и т.д.
  2. Padding (Внутренний отступ) — пространство вокруг содержимого.
  3. Border (Рамка) — граница вокруг внутреннего отступа.
  4. Margin (Внешний отступ) — пространство вокруг рамки, отделяющее элемент от других элементов.

Свойство display

Определяет, как элемент должен отображаться на странице и как он взаимодействует с другими элементами.

Основные значения:

  • block — блочный элемент. Занимает всю доступную ширину, начинается с новой строки. Можно задавать ширину и высоту. (Примеры: <div>, <p>, <h1>).
  • inline — строчный элемент. Занимает только необходимую ширину, не начинается с новой строки. Ширина и высота игнорируются (зависят от содержимого). Отступы (margin/padding) сверху и снизу могут работать некорректно. (Примеры: <span>, <a>).
  • inline-block — блочно-строчный элемент. Располагается в строке как строчный элемент, но позволяет задавать ширину, высоту и отступы как блочный элемент.
  • none — элемент полностью скрывается со страницы (не занимает места).
div {
    display: block;
}

span {
    display: inline-block;
    width: 100px;
    height: 50px;
}

Свойство padding (Внутренний отступ)

Определяет пространство между содержимым элемента и его рамкой. Внутренний отступ является частью самого элемента (фон элемента распространяется на padding).

Индивидуальные свойства

  • padding-top — отступ сверху
  • padding-right — отступ справа
  • padding-bottom — отступ снизу
  • padding-left — отступ слева

Сокращенная запись padding

Позволяет задать отступы для всех сторон сразу.

  • padding: 10px; — со всех 4 сторон 10px.
  • padding: 10px 20px; — сверху и снизу 10px, слева и справа 20px.
  • padding: 10px 20px 30px; — сверху 10px, слева и справа 20px, снизу 30px.
  • padding: 10px 20px 30px 40px; — сверху 10px, справа 20px, снизу 30px, слева 40px (по часовой стрелке).

Свойство border (Рамка)

Рамка располагается между внутренним (padding) и внешним (margin) отступами.

1. Стиль рамки (обязательное свойство)

Без указания стиля рамка не будет отображаться.

Свойство: border-style

Также существуют индивидуальные свойства: border-top-style, border-right-style, border-bottom-style, border-left-style.

Значения:

  • solid — сплошная линия
  • dashed — пунктирная линия
  • dotted — точечная линия
  • double — двойная линия
  • groove — объемная вдавленная линия
  • ridge — объемная выпуклая линия
  • inset — создает эффект вдавленного блока
  • outset — создает эффект выпуклого блока
  • none — без рамки
  • hidden — скрытая рамка

2. Ширина рамки

Свойство: border-width

Индивидуальные свойства: border-top-width, border-right-width, border-bottom-width, border-left-width.

Можно задавать в пикселях (px), em, rem или ключевыми словами: thin (тонкая), medium (средняя), thick (толстая).

3. Цвет рамки

Свойство: border-color

Индивидуальные свойства: border-top-color, border-right-color, border-bottom-color, border-left-color.

Принимает любые цветовые значения (hex, rgb, имя цвета, transparent).

Сокращенная запись border

Позволяет задать ширину, стиль и цвет одновременно (обычно для всех сторон).

/* Синтаксис: width style color */
.box {
    border: 2px solid red;
}

Также есть сокращения для отдельных сторон:

  • border-top
  • border-right
  • border-bottom
  • border-left
.box {
    border-bottom: 1px dashed #333;
}

Закругление углов border-radius

Свойство позволяет добавлять элементам закругленные углы.

Значения:

  • пиксели (px) — радиус скругления в пикселях.
  • проценты (%) — радиус скругления в процентах от размера элемента (50% для круга).

Можно задавать скругление для каждого угла отдельно:

  • border-top-left-radius — верхний левый угол
  • border-top-right-radius — верхний правый угол
  • border-bottom-right-radius — нижний правый угол
  • border-bottom-left-radius — нижний левый угол

Сокращенная запись:

/* Все углы 10px */
.box {
    border-radius: 10px;
}

/* Разные значения для углов */
.box {
    border-radius: 10px 20px 30px 40px; 
    /* верх-лев, верх-прав, низ-прав, низ-лев */
}

/* Создание круга */
.circle {
    width: 100px;
    height: 100px;
    border-radius: 50%;
}

Свойство margin (Внешний отступ)

Создает пустое пространство вокруг элемента, за пределами его рамки. Фон элемента НЕ распространяется на margin (он всегда прозрачный).

Индивидуальные свойства

  • margin-top — отступ сверху
  • margin-right — отступ справа
  • margin-bottom — отступ снизу
  • margin-left — отступ слева

Сокращенная запись margin

Работает аналогично padding.

  • margin: 10px; — со всех сторон 10px.
  • margin: 10px 20px; — сверху/снизу 10px, слева/справа 20px.
  • margin: 0 auto; — распространенный прием для центрирования блочного элемента по горизонтали (0 сверху/снизу, автоматически слева/справа).

Особенности margin

  • Схлопывание (Collapsing): Вертикальные отступы соседних элементов могут объединяться. Итоговый отступ равен большему из двух, а не их сумме.
  • Отрицательные значения: Margin может быть отрицательным, что позволяет сдвигать элементы или накладывать их друг на друга.
  • Значение auto: Браузер автоматически рассчитывает отступ. Часто используется для горизонтального центрирования.
.centered {
    width: 500px;
    margin: 0 auto; /* Центрирование блока */
}

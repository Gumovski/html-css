Изображения

Для вставки изображения в HTML используется тег <img>.

Атрибуты

  1. src
    путь к изображению
  2. alt
    текст, который будет показан, если изображение не загрузилось
  3. width
    ширина изображения. По умолчанию: ширина исходного изображения
  4. height
    высота изображения. По умолчанию: высота исходного изображения
<img src="image.png" alt="Изображение">

Карта изображения <map>

Позволяет создавать карту изображений с активными областями.

Карта изображения состоит из элементов <area>.

<img src="url" usemap="#имя_карты">
<map name="имя_карты">
...
</map>

Атрибуты элемента <area>

  1. shape
    форма области. Может принимать следующие значения:
    • rect - прямоугольник
    • circle - круг
    • poly - многоугольник
  2. coords
    координаты области.
    • для rect - координаты левого верхнего и правого нижнего углов
    • для circle - координаты центра и радиуса
    • для poly - координаты точек, которые соединяются линиями
  3. href
    ссылка, которая откроется при клике по области
  4. alt
    текст, который будет показан, если изображение не загрузилось
<img src="solar_system.jpg" alt="Солнечная система" usemap="#planetmap">

<map name="planetmap">
    <area shape="circle" coords="180,180,60" href="sun.html" alt="Солнце">
    <area shape="circle" coords="270,190,20" href="mercury.html" alt="Меркурий">
    <area shape="circle" coords="330,185,30" href="venus.html" alt="Венера">
    <area shape="circle" coords="390,185,35" href="earth.html" alt="Земля">
    <area shape="circle" coords="450,180,25" href="mars.html" alt="Марс">
</map>

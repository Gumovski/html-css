Изображения

Для вставки изображения в HTML используется тег <img>.

Атрибуты

  1. src
    путь к изображению
  2. alt
    текст, который будет показан, если изображение не загрузилось
  3. width
    ширина изображения. По умолчанию: ширина исходного изображения
  4. height
    высота изображения. По умолчанию: высота исходного изображения
<img src="image.png" alt="Изображение">

Карта изображения <map>

Позволяет создавать карту изображений с активными областями.

Карта изображения состоит из элементов <area>.

<img src="url" usemap="#имя_карты">
<map name="имя_карты">
...
</map>

Атрибуты элемента <area>

  1. shape
    форма области. Может принимать следующие значения:
    • rect - прямоугольник
    • circle - круг
    • poly - многоугольник
  2. coords
    координаты области.
    • для rect - координаты левого верхнего и правого нижнего углов
    • для circle - координаты центра и радиуса
    • для poly - координаты точек, которые соединяются линиями
  3. href
    ссылка, которая откроется при клике по области
  4. alt
    текст, который будет показан, если изображение не загрузилось
<img src="solar_system.jpg" alt="Солнечная система" usemap="#planetmap">

<map name="planetmap">
    <area shape="circle" coords="180,180,60" href="sun.html" alt="Солнце">
    <area shape="circle" coords="270,190,20" href="mercury.html" alt="Меркурий">
    <area shape="circle" coords="330,185,30" href="venus.html" alt="Венера">
    <area shape="circle" coords="390,185,35" href="earth.html" alt="Земля">
    <area shape="circle" coords="450,180,25" href="mars.html" alt="Марс">
</map>

CSS-свойства для изображений

Свойство width

Определяет ширину изображения.

Возможные значения:

  • auto - автоматическая ширина (по умолчанию)
  • пиксели - ширина в пикселях (например, 300px)
  • проценты - ширина в процентах от родительского элемента (например, 50%)
  • em, rem - относительные единицы измерения
  • vw - процент от ширины области просмотра (например, 50vw)
img {
    width: 300px;
}

img {
    width: 50%;
}

Свойство height

Определяет высоту изображения.

Возможные значения:

  • auto - автоматическая высота (по умолчанию)
  • пиксели - высота в пикселях (например, 200px)
  • проценты - высота в процентах от родительского элемента (например, 50%)
  • em, rem - относительные единицы измерения
  • vh - процент от высоты области просмотра (например, 50vh)
img {
    height: 200px;
}

img {
    width: 400px;
    height: auto; /* сохраняет пропорции */
}

Свойство clip-path

Создаёт область отсечения, которая определяет, какая часть изображения должна быть видимой.

Возможные значения:

  • none - без отсечения (по умолчанию)
  • circle() - круг. Синтаксис: circle(радиус at x y)
  • ellipse() - эллипс. Синтаксис: ellipse(rx ry at x y)
  • polygon() - многоугольник. Синтаксис: polygon(x1 y1, x2 y2, x3 y3, ...)
  • inset() - прямоугольник. Синтаксис: inset(top right bottom left round border-radius)
  • url() - путь к SVG-элементу <clipPath>
/* Круг */
img {
    clip-path: circle(50%);
}

/* Круг с центром в определённой точке */
img {
    clip-path: circle(40% at 50% 50%);
}

/* Эллипс */
img {
    clip-path: ellipse(50% 30% at 50% 50%);
}

/* Многоугольник (треугольник) */
img {
    clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
}

/* Прямоугольник со скруглёнными углами */
img {
    clip-path: inset(10px 20px 30px 40px round 15px);
}

Свойство filter

Применяет графические эффекты к изображению.

Возможные значения (функции фильтров):

  • none - без фильтра (по умолчанию)
  • blur(px) - размытие. Значение: радиус размытия в пикселях
  • brightness(%) - яркость. Значение: 0% (чёрный) до 100% (норма) и выше
  • contrast(%) - контрастность. Значение: 0% (серый) до 100% (норма) и выше
  • grayscale(%) - оттенки серого. Значение: 0% (норма) до 100% (полностью серое)
  • hue-rotate(deg) - поворот оттенка. Значение: угол от 0deg до 360deg
  • invert(%) - инверсия цветов. Значение: 0% (норма) до 100% (полная инверсия)
  • opacity(%) - прозрачность. Значение: 0% (прозрачный) до 100% (непрозрачный)
  • saturate(%) - насыщенность. Значение: 0% (ненасыщенный) до 100% (норма) и выше
  • sepia(%) - эффект сепии. Значение: 0% (норма) до 100% (полная сепия)
  • drop-shadow() - тень. Синтаксис: drop-shadow(x y blur color)

Можно комбинировать несколько фильтров:

/* Размытие */
img {
    filter: blur(5px);
}

/* Увеличение яркости */
img {
    filter: brightness(150%);
}

/* Чёрно-белое изображение */
img {
    filter: grayscale(100%);
}

/* Поворот оттенка */
img {
    filter: hue-rotate(90deg);
}

/* Инверсия цветов */
img {
    filter: invert(100%);
}

/* Тень */
img {
    filter: drop-shadow(5px 5px 10px rgba(0, 0, 0, 0.5));
}

/* Комбинация нескольких фильтров */
img {
    filter: brightness(110%) contrast(120%) saturate(130%);
}

/* Эффект сепии с увеличением контрастности */
img {
    filter: sepia(80%) contrast(110%);
}
